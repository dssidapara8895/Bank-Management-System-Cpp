# Bank Management System (C++)

A console-based banking application built in C++ using object-oriented programming.

## Features
- Account creation and secure login
- PIN authentication with account lock
- Deposit & withdrawal
- Money transfer between accounts
- Transaction history logging
- Admin panel for managing accounts
- File-based data storage

## Technologies
- C++
- File Handling
- OOP principles

## How to Run
g++ main.cpp Account.cpp BankSystem.cpp Utils.cpp -o bank
./bank
